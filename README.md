# XX电子商城系统
## 软件环境：linux、tomcat、JDK1.6、mysql等
## 开发工具：eclipse、Navicat for MySQL、tomcat7等
## 项目描述：该系统是为靓美衣舍淘宝店以及所有的小型企业做的一个轻量级电子商城，旨在建立一个以实现基本功能为基础，结构灵活、可复用、可扩展，开发快速且轻量的电子商务应用平台。为了能够让中小型企业特别是小型企业用上电子商务平台，我们特地开发此系统，使中小型企业也能做到生产，宣传，销售一条龙。该系统分为前台和后台两个系统，前台共有五个模块：
- 1.用户模块：包括用户的注册、激活、登录和退出等功能。系统中用户的注册需要通过前台JS的格式校验、用户名是否存在的AJAX异步校验和后台Struts2校验，核对验证码无误，才能完成注册，发送用激活邮件激活账号； 
- 2.分类模块?：查询一级分类、二级分类并显示；
- 3.商品模块：包括最新商品、热门商品、显示二级分类商品和显示商品详情等功能；
- 4.购物模块：?包括添加到购物车、清空购物车和从购物车中移除商品等功能；
- 5.订单模块:：包括生成订单、为订单付款、查询我的订单和订单详情等功能。系统中的为订单付款包括在线支付功能、修改订单状态和修改订单信息等，其中在线支付功能当前主要是实现了网银支付功能。后台也分为五个模块，分别为对用户模块、一级分类、二级分类、商品模块和订单模块的管理，主要是实现对各自对应数据表的增删改查的操作。
